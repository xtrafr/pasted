# Pasted

Pasted keeps links, notes and reminders in one place. Drop in a messy text file, choose what is worth saving and leave the rest behind.

The application is under active construction. Full setup, architecture, screenshots and self-hosting instructions will be added as each working phase lands.

## Local development

```bash
pnpm install
pnpm dev
```

Copy `.env.example` to `.env` before using authenticated features.
